Add slideshow photos here using this naming pattern:

slide-1.jpg
slide-2.jpg
slide-3.jpg

You can also use:
.jpeg
.png
.webp

The website will automatically load them in numbered order.

To change the slideshow speed, edit:
`data-speed="3400"`

inside the `Snapshots` card in:
